媒体查询(@media)与响应式布局

上面这个东西我也看不懂，但是如果要做到响应式布局就需要根据不同的屏幕大小使用不同的布局和样式

介绍 @media screen !!!

它可以帮助你在不同的屏幕尺寸中使用不同的布局和样式

这很“简单”!

min-width:value则表示当屏幕宽度>value时应用其中的css

max-width:value则表示当屏幕宽度<value时应用其中的css

其中的screen and不知道是什么意思，反正我删掉好像也没什么影响

@media (max-width:991px){
   .aside{
    display: none;
   }
.section{
    width: 100%;
    }
}

上面这段代码表示，在屏幕宽度小于991px时，将侧边的导航栏(.aside)的显示方式设置为none，并将页面主体(.section)的宽度设置为100%。

效果图

不仅仅是显示方式和宽度，你可以设置和修改任何你想在不同屏幕宽度下应用的样式，它很灵活。

值得注意的地方

@media screen 不仅仅可以用于不同设备的响应式布局，还可以应用于网页大小调整时对页面元素的属性进行调整，使页面在任何宽度下都能保持其结构，但这往往需要设置多个阈值，这时就需要注意多个媒体查询在css中的设置顺序，顺序错误会使有些媒体查询的效果被覆盖，这个我也解释不清楚，总之如果效果有问题可以试着自己调整顺序然后调试(好屑啊这个作者）
